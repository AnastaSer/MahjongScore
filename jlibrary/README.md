# Mahjong Hand Score Library

Библиотека для расчёта стоимости одной собранной игроком руки.

## 🚀 Возможности
 - Учёт ветра игрока и преимущественного
 - Расчёт по перечню комбинаций 
 - Расчёт по перечню костей (библиотека сама собирает из них комбинации с учётом порядка)
 - Три варианта правил: 
   - классические,
   - вариант друзей с увеличенным коэфициентом за чистую масть и без лимита
   - вариант родителей (разрешение на составление 4х костей в последовательность)

## 📦 Требования

- Java 21+
- Gradle (или просто указать, что сборка через ./gradlew)

## 🔧 Сборка и запуск

### Библиотека подсчёта очков

cd jlibrary
./gradlew clean build
java -jar build/libs/hand_scoring-1.0.jar [аргументы]

Пример использования

java -jar hand_scoring-1.0.jar восточный открытая пара дракон зеленый закрытая тройка дракон красный закрытая четверка дракон белый закрытая тройка ветер восточный закрытая четвёрка ветер северный правила:друзья

## 📖 Формат входных данных

### Ветра: 
 -"east", "восток", "восточный";
 - "south", "юг", "южный";
 - "west", "запад", "западный";
 - "north", "север", "северный".
 
### Драконы:
 - "white", "белый";
 - "red", "красный";
 - "green", "зелёный", "зеленый".
 
### Комбинации и кости начинаются с признака "открытый/закрытый":
 - "opened", "открытая", "открытый";
 -  "closed", "закрытая", "закрытый".
 
### Комбинации:
 - "one", "single", "один", "одна";
 - "pair", "two", "пара", "два", "две";
 - "three", "панг", "тройка", "три";
 - "four", "конг", "четверка", "четвёрка", "четыре";
 - "ordered", "чоу", "последовательность", "подряд" с уточнением:
   - "three", "три", "база", "классика";
   - "four", "четыре", "расширение".

### Кости:
 - "sign", "знак", "символ";
 - "dot", "дот", "точка";
 - "bamboo", "бамбук";
 - "wind", "ветер";
 - "dragon", "дракон";
 - "bonus", "бонусный", "бонус" с уточнением:
   - "season", "сезон";
   - "flower", "цветок".

И номинал: число от 1 до 9 для масти, либо от 1 до 4 для бонусных костей,
           слово для обозначения ветра или дракона.

### Флаги:
 - чистая масть;
 - чистая масть с драконами и ветрами;
 - только драконы и ветра;
 - драконы, ветра, единицы и девятки;
 - маджонг;
 - мизер;
 - без последовательностей;
 - все драконы;
 - все ветра;
 - предмаджонг со старта;
 - завершение единственной возможной;
 - завершение костью со стены;
 - завершение свободной костью;
 - завершение последней доступной;
 - завершение ограблением открытого конга.
 
## ⚙️ Настройка правил

Правила хранятся в src/main/resources/*.rules в формате:
properties

doubles.clear.suit=3
mahjong.score=20
max.one.hand.score=2000

Доступные наборы: classic.rules, parents.rules, friends.rules
